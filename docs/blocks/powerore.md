# Power Ore

Power Ore is and ore that generates in the overworld in [Synthesis Ore Veins](../worldgen/synthesisores) that drops a random number of Power Materials between 1 and 3 the items that drop are listed below.
There is an [End Stone](https://minecraft.gamepedia.com/End_Stone) version of this ore.

# Drops
* [Power Shard](../items/synthesismaterials)
* [Power Stone](../items/synthesismaterials)
* [Power Gem](../items/synthesismaterials)
* [Power Crystal](../items/synthesismaterials)
