# Rare Prize Blox

Rare Prize Blox are Bug Blox that can be found in [Bug Blox Veins](../worldgen/bugbloxveins). Destroying a Rare Prize Blox will drop a random number of items (randomly selected from a pool of drops) between 1 and 3 the items that are dropped are [listed below](#drops)
